# uuuSanAI — Claude Code 작업 지침

이 디렉터리는 사용자의 AI 에이전트 시스템 리서치 프로젝트입니다.

## 세션 재개 시 먼저 할 일

사용자가 이 프로젝트에서 작업 재개 신호를 주면 (예: "어제 이어서", "Phase 2 가자", "graft 진행", "이어서 하자") **가장 먼저 다음 파일을 읽으세요**:

- `meta/next-session-handoff.md` — Phase 진행 상황, 다음 작업 결정 지점, 즉시 실행 가능 항목

이 파일은 세션 종료 시점에 갱신됩니다. 상단 `date_recorded` 로 신선도 확인.

새로운 독립 요청(별도 리서치 등)이면 이 파일 무시 가능.

## 프로젝트 개요

- **목적**: LLM/에이전트 동향, 하네스 엔지니어링, 에이전트 프로덕트 리서치
- **Phase 1**: 편향 없는 외부 웹 리서치. 사용자의 다른 프로젝트 상태가 분석을 편향시키지 않도록, Phase 1 중에는 사용자 프로젝트에 대해 먼저 묻거나 가정하지 말 것.
- **Phase 2**: 사용자가 직접 연결하자고 할 때 graft-evaluator 로 자기 프로젝트와 합류.
